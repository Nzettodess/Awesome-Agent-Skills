# Awesome Agent Skills

A curated collection of specialized skills designed to extend the capabilities of AI agents. These skills provide modular functionality for various domains, from data analysis and visualization to frontend development and reporting.

## 📚 Skills Catalog

### 🤖 Agentic AI Development

Enhance agent reasoning, evaluation, and context management.

- **agent-evaluation**: Tools for assessing agent performance.
- **context-engineering**: Techniques for optimizing context window usage.
- **llm-evaluation**: Frameworks for evaluating Large Language Models.
- **prompt-engineering**: Best practices and templates for effective prompting.
- **thought-based-reasoning**: Modules to improve chain-of-thought capabilities.
- **AI-research-SKILLs**: Comprehensive AI research and analysis tools.

### 📊 Charts

Generate professional visualizations and diagrams.

- **mermaidjs-v11**: Create diagrams using Mermaid.js syntax.
- **plotly**: Interactive scientific and statistical charts.
- **seaborn**: Statistical data visualization based on matplotlib.

### 📈 Data Analytic

Perform complex data analysis and engineering tasks.

- **excel-analysis**: Automated analysis of Excel spreadsheets.
- **senior-data-engineer**: Advanced data pipeline and engineering workflows.
- **senior-data-scientist**: Statistical modeling and machine learning capabilities.
- **xlsx**: Low-level Excel file manipulation.

### 💻 Development

Tools for software development and code quality.

- **code-review**: Automated code review and assistance tools.

### 🎨 Frontend

Design and prototype user interfaces.

- **canvas-design**: HTML5 Canvas drawing and manipulation.
- **frontend-ui-ux**: UI/UX design patterns and component generation.
- **ui-ux-pro-max**: Design Intelligence - Searchable database of UI styles, color palettes, and UX guidelines.

### 📑 Reports

Generate high-quality documents and presentations.

- **notebooklm-slides**: Convert notebook content into presentation slides.
- **pdf-generator**: Create professional PDF documents programmatically.

### 🛠️ Skill Creator

- **Claude Skill Creator**: Tools for building skills specifically for Claude.
- **One4All Skill Creator**: General-purpose skill creation and management.
